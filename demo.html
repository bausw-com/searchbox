Try: pagefind, search, indexing, multilingual

Searchbox is a UI component built on top of Pagefind providing fast, backend-less search to any website.

To get started, please follow the Pagefind's instructions on building an index for your website.

Usage

        <!-- 1. Create your search input -->

<div id="search">
  <input type="search" />
</div>

<!-- 2. Add searchbox.js -->

<script type="module" src="/searchbox.min.js"></script>

<!-- 3. Create Searchbox instance -->

<script>
const searchbox = new Searchbox({
  props: {
    options: {
      pagefind: 'https://pagefind.app/_pagefind/pagefind.js',
    },
  },
  target: document.querySelector('#search')
})
</script>
